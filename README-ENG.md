# **Game Store – Online Video Game Shop (HTML + Custom CSS)**

A **modern, elegant, and 100% responsive** website for selling video games, built with **pure HTML and custom CSS** (no heavy frameworks)!

### **What this page does (and why it's awesome)**

It is a **complete video game store** in landing-page style, with everything a gamer needs:

* **Header with hamburger menu** that automatically becomes mobile-friendly (using only a checkbox!)
* **Logo + impactful title** with action buttons
* **Popular games section** with 9 images in a row
* **Full catalog** with **18 real video games** (Call of Duty, FIFA 25, Elden Ring, GTA V, etc.)

  * Each game includes: image, name, price, and “Buy” button
  * Real and updated prices
* **Subscription form** for newsletter
* **Complete footer** with:

  * Logo + social media (Facebook, WhatsApp, Twitter/X, Instagram, Twitch)
  * 2 columns of useful links
  * Professional dark design

### **Technical features (PRO level)**

* **Responsive menu without JavaScript** → uses only HTML + CSS (checkbox hack)
* Clean, well-structured, and well-commented code
* Fully **responsive** (looks perfect on mobile, tablet, and desktop)
* Real images of current games (2024–2025)
* Modern gamer-style **dark mode** design
* Buttons with hover and visual effects
