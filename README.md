# Game Store – Tienda de Videojuegos Online (HTML + CSS Personalizado)

¡Una página web **moderna, elegante y 100% responsive** para vender videojuegos, hecha con **HTML puro y CSS personalizado** (sin frameworks pesados)!

### Qué hace esta página (y por qué es increíble)

Es una **tienda de videojuegos completa** tipo landing page con todo lo que necesita un gamer:

- **Header con menú hamburguesa** que se convierte en móvil automáticamente (¡usando solo un checkbox!)
- **Logo + título impactante** con botones de acción
- **Sección de juegos populares** con 9 imágenes en fila
- **Catálogo completo** con **18 videojuegos reales** (Call of Duty, FIFA 25, Elden Ring, GTA V, etc.)
  - Cada juego tiene: imagen, nombre, precio y botón "Comprar"
  - Precios reales y actualizados
- **Formulario de suscripción** para newsletter
- **Footer completo** con:
  - Logo + redes sociales (Facebook, WhatsApp, Twitter/X, Instagram, Twitch)
  - 2 columnas de enlaces útiles
  - Diseño oscuro profesional

### Características técnicas (nivel PRO)
- **Menú responsive sin JavaScript** → usa solo HTML + CSS (checkbox hack)
- Código limpio, bien estructurado y comentado
- Totalmente **responsivo** (se ve perfecto en celular, tablet y PC)
- Imágenes reales de juegos actuales (2024-2025)
- Diseño moderno tipo "dark mode" gamer
- Botones con hover y efectos visuales

### Ideal para
- Proyectos finales de desarrollo web
- Portafolio profesional (¡esto impresiona muchísimo!)
- Tiendas reales de videojuegos
- Aprender HTML + CSS desde cero
